# Dictionary

This repository is Persian Dictionary with jetpack compose and use local Datbase from assets

Features
+ Display all words.
+ save last seen words.
+ show Examples and definition.
 

## Libraries used

+ Jetpack compose
+ paging 3
+ navigation animation
+ coroutines
+ Flow
+ Room
